  • General Information

    • When we develop an algorithm, we usually need to manipulate some data. The data can be a single value, a collection of values, or other kinds. The way to store these data in a computer is called data structure. We can think of a variable (named memory location) as the most basic data structure. In this course, other data structures will be studied in the context of abstract data types (ADT): a specification of data and set of operations on the data. We will also analyze the efficiency (in terms of time and space) of the operations implemented in the ADT's.
