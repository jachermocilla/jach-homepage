  • Aim

    • Organizations across every sector are delivering their functions and services via sophisticated web-based services. Web developers are increasingly deploying customized and powerful server-side software applications for use. This area of online development is generally referred to as web application development or Internet application programming. This course teaches the concepts and techniques for developing advanced server-side and client-side web functionality. Students will learn the skills and methods to effectively program and integrate software to deliver robust web-based solutions.

  • Objectives

    • At the end of this course the students should be able to:

    1. analyze web application requirements;
    2. discuss ethical implications of web-based applications;
    3. design web pages;
    4. write applications for mobile internet; and
    5. develop and implement web-based applications.
