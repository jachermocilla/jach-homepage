  • Aim

    • The aim of this course is to introduce students to data communications and networking concepts.

  • Objectives

    • At the end of this course the students should be able to:

    1. explain how data is transmitted over communication channels;
    2. identify and describe the different layers of the OSI reference model and TCP/IP protocol suite, including the protocols available for each layer;
    3. understand the architecture of Local Area Networks and Internetworks and the services they provide;
    4. understand the importance of security in a networked environment;
    5. develop network-enabled applications using a high-level programming language;
    6. and perform basic network administration.

  • Topics (tentative)

    1. Computer Networks Overview
      1. Introduction
      2. Protocols and Architecture
    2. Data Communications
      1. Data Transmission
      2. Transmission Media
      3. Data Encoding
      4. Data Link Control
    3. Network Hardware Components
      1. Layer 1 and Layer 2 devices
    4. Local Area Networks
      1. LAN Technology
      2. LAN Systems
    5. Communication Architecture and Protocols
      1. Internetworking and Network Layer Concepts
      2. Transport Layer
      3. Application Layer
      4. Network Security
