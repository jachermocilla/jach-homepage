  • Aim

    • The aim of this course is to introduce students to the fundamentals of programming using the C programming language.

  • Objectives

    • At the end of this course the students should be able to:

    1. create well-structured programs proficiently in C programming language;
    2. design and implement recursive algorithms;
    3. use files and pointers in programming; and,
    4. prove the correctness of a program.

  • Topics (tentative)

    1. Introduction
      1. The Program Development Process
      2. Structured Programming
    2. Functions in C
      1. Parameter Passing
        1. Pass by Value
        2. Pass by Reference
        3. Pointers
        4. Arrays and Strings
        5. Structure and Array of Structures
      2. Recursion
    3. Linked Lists
      1. Singly Linked Lists
      2. Doubly Linked Lists
      3. Dummy Nodes
      4. Circular Linked Lists
    4. Files
      1. Text and Binary Files
      2. Loading Data from Files
      3. Saving Data into Files
    5. Program Correctness
      1. Unit Testing
      2. Using Loop Invariant
