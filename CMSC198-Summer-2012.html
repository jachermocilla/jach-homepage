CMSC 198 Practicum

  • Deliverables

    • Software Requirement Specification (SRS): This document describes the functional and non-functional requirements of the software that the student will work on. Both the faculty adviser and company supervisor should agree on the contents of this document.
    • Daily Activity Log Notebook (DALN): The daily activity log notebook contains daily entries describing the assigned task(s) for the day. It also contains a description of how the student went about finishing the task(s), describing the challenges faced and the solutions provided. Each entry should be signed by the company supervisor.
    • Final Report: This document contains a summary of the SRS and DALN and a description of the completed project. The description should include screenshots of the final output as well as important design artifacts like use case diagrams, class diagrans, ERDs, flow charts, etc. In the final report, the student should also include a description of the overall experience in the OJT.
    • Weekly Email: Email your faculty supervisor about your weekly accomplishments. The email should be sent on a Friday.
    • Company Evaluation Form: Evaluation form accomplished by the company supervisor.

  • Grading

    • Company Supervisor Evaluation - 80%
    Faculty Adviser Evaluation - 20%
    Total(Final Grade) - 100%

  • Some Tips!

    • Work like you are already a professional.
    • Report to work on time. Leave work on time.
    • Try to learn on your own but do not be shy to ask for help.
    • Be respectful to co-workers.
    • Don't be a complainer.
    • Participate in social activities.
    • Make a plan for each day.
